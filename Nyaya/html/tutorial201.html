Syntax Definition

A formal grammar (sometimes simply called a grammar) is a set of formation rules for strings in a formal language. The rules describe how to form strings from the language's alphabet that are valid according to the language's syntax. A grammar does not describe the meaning of the strings or what can be done with them in whatever context – only their form. [Wikipedia].

We‘ve already learned that propositional formulas are build form atoms ‘p,q,r,…’ and connectives ‘¬∧∨→’.

atoms p, …
negation ¬ ¬p “not p”
conjunction ∧ p ∧ q “p and q”
disjunction ∨ p ∨ q “p or q”
implication → p → q “if p then q”

But it was a informal introduction only. We have to define what a “well formed formula” in propositional logic looks like. This definition is done with a formal grammar. Parenthesis are used to avoid ambiguities.

P  ::=  p  |  (¬P)  |  (P ∧ P)  |  (P ∨ P)  |  (P → P)

A propositional formula ‘P’ (on the left side) is defined ‘::=’ through the expression on the right side. The expression of the right side consists of serveral parts, devided by ‘|’, which separates alternatives for P. So ‘P’ is either and atom ‘p’, or the negation of a formula ‘(¬P)’, or the conjunction of two formuals ‘(P ∧ P)’, or the disjunction of two formuals ‘(P ∨ P)’, or the implication of two formuals ‘(P → P)’.

There is a inaccuracy in this definition. Only the symbol ‘p’ is defined for atoms. We use ‘p’ as representative for symbols {a,b,…,p,q,r,…,p1,p2,…} we want to use for atoms. In a complete formal grammar the forms of atomes have to be defined, which would distract from the core of the grammar above. A propositional formula can be build from other propositional formulas recursively. Atoms terminate the recursion.

Example

To create a well formed formula, we start with “P” and replace it with “(P ∧ P)” (1). Then we substitute the first “P” with “(P ∨ P)” (2) and the second “P” with “(¬P)” (3). After that we replace the seconde “P” with “(P → P)”. …

P ➜1 (P ∧ P) ➜2 ((P ∨ P) ∧ P) ➜3 ((P ∨ P) ∧ (¬P)) ➜4 ((P ∨ (P → P)) ∧ (¬P))
  ➜5 ((P ∨ (P → (¬P))) ∧ (¬(P ∨ P))) ➜6 … ➜10 ((p ∨ (q → (¬p))) ∧ (¬(q ∨ p)))

We observe that a lot of parenthesis are produced.

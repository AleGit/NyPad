Valuation

In the previous chapter we‘ve defined the syntax of propositional formula, but we‘ve ignored that propositions can be true or false. We didn‘t attach any meaning to our formulas. We didn‘t define any semantics of propositional logic. This will be done now.

The set of variables of a formula is the set of sub-formulas of the formula, where the elements are atoms. The set of variables of the formula a ∨ ¬b ∧ c → a is {a,b,c}. A valuation or truth assingment is a mapping from the set of variables to the set of truth values {true, false}. We will use “T” as abbreviation for “true” and “F” as abbreviation for “false”.

v(a) = F     v(b) = F     v(c) = T

We extend the valuation to formulas recursivly for composite formulas.

  • v(¬P) = T if and only if v(P) = F. Otherwise v(¬P) = F.
  • v(P∧Q) = T if and only if v(P) = T and v(Q) = T. Otherwise v(P∧Q) = F.
  • v(P∧Q) = F if and only if v(P) = F and v(Q) = F. Otherwise v(P∨Q) = T.
  • v(P→Q) = F if and only if v(P) = T and v(Q) = F. Otherwise v(P→Q) = T.

Example

  • v(a) = F   assingment
  • v(b) = F   assingment
  • v(c) = T   assingment
  • v(¬b) = T   because v(b) = F.
  • v(¬b∧c) = T   because v(¬b) = T and v(c) = T.
  • v(a∨(¬b∧c)) = T   because v(¬b∧c) = T.
  • v((a∨(¬b∧c))→a) = F   because v(c) = T and v(a) = F.

We transfer the valuation from the atoms to the sub-formulas until a value is assigned to the formula. For clarity we can write down the results in a table with two rows. In the first row we write down all sub-formulas (including variables and the formula itself). In the second row we fill in the values of the variables. Then we fill in the values of composite sub-formulas step by step.

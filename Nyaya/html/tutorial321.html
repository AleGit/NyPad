Semantic Entailment

For propositional formulas P1, P3, …, Pn, Q and the semantic entailment relation ⊨, we define the semantic entailment

P1, P3, …, Pn ⊨ Q

A semantic entailment holds, if for all valuations which evaluate all Pi on the left side to true, Q is evaluated to true too.

We can use a truth table to check if a semantic entailment holds.

  • a ⊨ ¬a   does not hold
  • a, b ⊨ a ∧ b   holds
  • a ⊨ a ∧ b   does not hold
  • a ∧ b ⊨ a ∨ b   holds
  • a ∨ b ⊨ a ∧ b   does not hold
  • b ⊨ a → b   holds

Semantic Equivalence

For propositional formulas P and Q, where the semantic entailment P ⊨ Q holds and the semantic entailment Q ⊨ P holds, we say P and Q are semantically equivalent.

P ≡ Q

You have to check for all cases where the left side evaluates to true, if the right side evaluates to true too. Ditto you have to check for all cases where the right side evaluates to true, if the left side evaluates to true too. If both holds, than the left and right side are semantically equivalent.

  • a ≡ ¬¬a
  • a ∧ b ≡ b ∧ a
  • a ∨ b ≡ b ∨ a
  • a → b ≢ b → a
  • ⊥ ≡ a ∧ ¬a
  • ⊤ ≡ a ∨ ¬a

Sub-formulas can be substituted with equivalent formulas, because for the same truth assingment they evaluate to the same truth value. So they do not change the evaluation of a formula and does not affect satisfiability or validity.

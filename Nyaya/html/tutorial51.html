Boolean Functions

In general a function is a mapping from one set of objects to another set of objects. The natural sum function on n arguments is a mapping from ℕn (set of n-tuples) to ℕ (set of natural numbers). The sum function with three arguments maps (4,8,100) to 112 and we write sum(4,8,100) = 112.

A Boolean function on n arguments is a mapping from {0,1}n to {0,1}.

There are two different Boolean functions with zero arguments – the first one maps every element of {()} to 0, the second one to 1. There are four different Boolean functions with one argument and 16 different Boolean functions with 2 arguments.

In general there are 22n different Boolean functions with n arguments.

Representations
Usually the complement of «x» is notated as «x». This is impractical for text input and «!x» or «~x» are often used instead.

The product «•» maps two arguments to «1», if both arguments are equal to «1». Otherwise (if one or two arguments are «0») the product «•» maps its arguments to «0».

The sum «+» maps two arguments to «0», if both arguments are equal to «0». Otherwise (if one or two arguments are «1») the sum «+» maps its arguments to «1».

The exclusive or «⊕» maps two arguments to «1», if the arguments are different. Otherwise (if the arguments are equal) the exclusive or «⊕» maps its arguments to «0».

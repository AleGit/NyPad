Natural deduction (short overview)

We‘ve defined how well formed formula looks like in propositional logic. But we still can‘t proove anything, we still can‘t infer anything from a formula. Natural deduction is a collection of proof rules, which can be used to proof the validity of sequents.

Basic rules

We write down basic rules as valid sequents, which we don‘t have to proove anymore.

New Symbols

⊤ = p ∧ ¬p     ⊥ = p ∨ ¬p

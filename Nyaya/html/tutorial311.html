When we write down all possible truth assignments of a formula and we calculate the truth value of the formula for each assignment we create the truth table of the formula.

There are two different truth assignments for one variable and so there are 2×2=4 different truth assignments for two variables.

In general there are 2n different truth assignments for n variables. Thus the calculation of a truth table for a formula with many variables is a very expensive operation, because for each additional variable the calculation time will double.

Although truth tables are unpractical for “big” formulas, they are are useful do demonstrate some properties of “small” formulas.

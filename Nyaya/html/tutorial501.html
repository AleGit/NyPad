Boolean Functions

In general a function is a mapping from on set of objects to another set of objects. For example the natural sum function on n arguments is a mapping from {1,2,3,…}n to {1,2,3,…}. The sum function with three arguments maps 4,8,100 to 112 and we write sum(4,8,100)=112.

A Boolean function on n arguments is a mappging from {0,1}n to {0,1}.

For example g(0,1) = 1 is a Boolean function which maps the two arguments 0 and 1 to the result 1.

A Boolean function with zero arguments is actually a constant – either 0 or 1. So there are two different Boolean functions c() with zero arguments. You can construct four different Boolean functions f(a) with one argument. 16 differnt Boolean functions g(a,b) with 2 arguments. 256 different Boolean functions h(a,b,c) with 3 arguments. In general we can construct are 22n different Boolean functions with n arguments.

Representations

Propositional formulas and truth tables are different representations of Boolean functions.

Does a.(b+c) = a.b+a.c look right, but a+b.c = (a+b).(a+c) does not? Don‘t mix up boolean expressios with arithmetic expressions. The second equivalence holds too! You can check it with a truth table.

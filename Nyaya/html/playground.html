Adding syntax trees:

  • Tapping on the button “Formulas“ at the top left and then tapping on one of the formulas in the list will add the syntax tree of the tapped formula to the playground.
  • Tapping and holding a free space on the playground will add the syntax tree of the simple formula p to the playground.

Interacting with Syntax trees:

  • Tapping into a syntax tree will (de)select the syntax tree.
  • Tapping into a syntax tree and moving will move the syntax tree.
  • Tapping the lock symbol at the top left of a selected syntax tree will (un)lock the syntax tree.
  • Tapping the delete symbol at the top right of a selected syntax tree will delete the syntax tree.
  • A syntax tree is in free mode, when the lock symbol is open.
  • In free mode the syntax tree can be manipulated freely.
  • Tapping a node will offer a list of actions.
  • Atoms can be assigned to red (true), green (false) or blue (unassigned). Connectives will evaluate to red (true), green (false) or blue (pendant) automatically.
  • A syntax tree is in locked mode when the lock symbol is closed.
  • Tapping a node will offer a list of equivalence transformations.

• Adding a syntax tree from the list of formulas: Tap on the button “Formulas“, tap on one formula in the list. The syntax tree for the selected formula will be added to the playground.
  • Adding a new syntax tree to the playground: Tap and hold on a place in the grid. A new syntax tree with simple formula “p” will be added to the playground on the tapped place.
  • Selecting a syntax tree: Tap on a syntax tree.
  • Moving a syntax tree: Tap and move a syntax tree.
  • Locking or unlocking a syntax tree: Tap the lock symbol.
  • Deleting a syntax tree: Tap the red symbol.

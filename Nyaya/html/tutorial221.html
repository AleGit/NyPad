When we use the grammar rule “P1 ➜ (P2 ∨ P3)” we build the more complex formula “P1” from two less complex formulas “P2” and “P3”. “P2” and “P3” are called subformulas of “(P2 ∨ P3)”. All subformulas of “P2” are subformulas of “(P2 ∨ P3)”, all subformulas of “P3” are subformulas of “(P2 ∨ P3)” and “(P2 ∨ P3)” is a subformula of “(P2 ∨ P3)” too. Per Definition atoms are the smallest subformulas of every formula.

From subformulas to formulas

If you are using formulas in strict syntax, you can use any rule of the grammar to combine them to more complex formulas without additional work. But if you are using the syntax conventions, you have to be carefull when you are combining subformulas.

For example if you combine the formulas a ∨ b and c → d with the rule P ➜ (P∧P) to a more complex formula then the correct expression (after ommitting the outer parenthesis) is  a ∨ b ∧ c → d  (a ∨ b) ∧ (c → d). You can verify this by rewriting the two subformulas to formulas in strict syntax.

1. a, b P ➜ (P ∨ P) (a ∨ b) a ∨ b
2. c, d P ➜ (P → P) (c → d) c → d
3. (a ∨ b), (c → d) P ➜ (P ∧ P) ((a ∨ b) ∧ (c → d)) (a ∨ b) ∧ (c → d)
4. ((a ∨ b) ∧ (c → d)) P ➜ (¬P) ¬((a ∨ b) ∧ (c → d)) ¬((a ∨ b) ∧ (c → d))

From formula to subformulas

To determine the sub-formulas of a formula you rewrite the formula in strict syntax. Your remove the outermost parentheses and you split the formula by be removing that connective, which leaves one or two well formed subformulas.

( (a ∨ b) ∧ (c → d) )    ( (a ∨ b) ∧ (c → d) )    ( (a ∨ b) ∧ (c → d) )

Then your original formula must be reconstructable with one grammar rule from the extracted subformulas.

(a ∨ b)    (c → d)   •  P ➜ (P ∧ P)  •  (a ∨ b) ∧ (c → d)

When we use the grammar rule “P1 ➜ (P2 ∨ P3)” we build the more complex formula “P1” from two less complex formulas “P2” and “P3”. “P2” and “P3” are called subformulas of “(P2 ∨ P3)”. All subformulas of “P2” are subformulas of “(P2 ∨ P3)”, all subformulas of “P3” are subformulas of “(P2 ∨ P3)” and “(P2 ∨ P3)” is a subformula of “(P2 ∨ P3)” too. Per Definition atoms are the smallest subformulas of every formula.

From subformulas to formulas

If we are using formulas in strict syntax, we can use any rule of the grammar to combine them to more complex formulas without additional work. But if we are using the syntax conventions, we have to be carefull when we are combining subformulas.

For example if we combine the formulas a ∨ b and c → d with the rule P ➜ (P∧P) to a more complex formula then the correct expression (after ommitting the outer parenthesis) is  a ∨ b ∧ c → d  (a ∨ b) ∧ (c → d). We can verify this by rewriting the two subformulas to formulas in strict syntax.

3. (a ∨ b), (c → d) P ➜ (P ∧ P) ((a ∨ b) ∧ (c → d)) (a ∨ b) ∧ (c → d)
4. ((a ∨ b) ∧ (c → d)) P ➜ (¬P) ¬((a ∨ b) ∧ (c → d)) ¬((a ∨ b) ∧ (c → d))

From formula to subformulas

To determine the sub-formulas of a formula we rewrite the formula in strict syntax. We remove the outermost parentheses and we split the formula by be removing the root connective, which leaves one (negation) or two (conjunction, disjunction, implication) well formed subformulas.

( (a ∨ b) ∧ (c → d) )    ( (a ∨ b) ∧ (c → d) )    ( (a ∨ b) ∧ (c → d) )

Then our original formula must be reconstructable from the extracted subformula(s) with the grammar rule for the removed connective.

(a ∨ b)    (c → d)   •  P ➜ (P ∧ P)  •  (a ∨ b) ∧ (c → d)

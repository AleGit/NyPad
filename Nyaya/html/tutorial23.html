When we use the grammar rule “P1 ➜ (P2 ∨ P3)” we build the more complex formula “P1” from two less complex formulas “P2” and “P3”. “P2” and “P3” are called sub-formulas of “(P2 ∨ P3)”. All sub-formulas of “P2” are sub-formulas of “(P2 ∨ P3)”, all sub-formulas of “P3” are sub-formulas of “(P2 ∨ P3)” and “(P2 ∨ P3)” is also a sub-formula of “(P2 ∨ P3)”. Per definition atoms are the smallest sub-formulas of every formula.

From sub-formulas to formulas

If we are using formulas in strict syntax, we can use any rule of the grammar to combine them to more complex formulas without additional work. But if we are using the syntax conventions, we have to be careful when we are combining sub-formulas.

For example if we combine the formulas a ∨ b and c → d with the rule P ➜ (P∧P) to a more complex formula then the correct expression (after omitting the outer parenthesis) is  a ∨ b ∧ c → d  (a ∨ b) ∧ (c → d). We can verify this by rewriting the two sub-formulas to formulas in strict syntax.

4. ((a ∨ b) ∧ (c → d)) P ➜ (¬P) ¬((a ∨ b) ∧ (c → d)) ¬((a ∨ b) ∧ (c → d))

From formula to sub-formulas

To determine the sub-formulas of a formula we rewrite the formula in strict syntax. We remove the outermost parenthesis and we split the formula by removing the root connective, which leaves one (in the case of a negation) or two (conjunction, disjunction, implication) well formed sub-formulas.

( (a ∨ b) ∧ (c → d) )    ( (a ∨ b) ∧ (c → d) )    ( (a ∨ b) ∧ (c → d) )

Then our original formula must be reconstructable from the extracted sub-formula(s) with the grammar rule for the removed connective.

(a ∨ b)    (c → d)   •  P ➜ (P ∧ P)  •  (a ∨ b) ∧ (c → d)

Chapter 3
Semantics

Properties

Satisfiability

A propositional formula P is satisfiable,
if there is at least one truth assignment that v(P) = T.
A propositional formula P is not satisfiable,
if there is no truth assignment that v(P) = T.

Validity

A propositional formula P is valid,
if v(P) = T for every possible truth assignment.
A propositional formula P is not valid,
if there is at least one truth assignment, that v(P) = F.

Examples

  • ¬a, a∧b, a∨b, a→b are satisfiable, but not valid.
  • ⊥, a∧¬a are not satisfiable. Hence they are not valid.
  • ⊤, a∨¬a, (p∧¬q→r)∧(¬r)∧(p)→q are valid. Hence they are satisfiable.

For these and similar simple examples satisfiability and validity can be checked by using truth tables. Due the exponential growing costs of computing truth tables this is not a practical way.

We can save some calculation time with smarter valuation. This will help some of the time, but truth tables stay unpractical for “big” formulas.
